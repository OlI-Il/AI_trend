Fetch today's trending AI videos and generate a report.

## Instructions

1. Use the YouTube MCP server to search for the top 10 trending AI videos from the last 24 hours. Search for topics like: "artificial intelligence", "machine learning", "LLM", "AI agents", "generative AI".

2. For each video, collect:
   - Title
   - Channel name
   - View count
   - Published date
   - URL
   - Brief description

3. Generate a clean markdown report and save it to `output/{{ YYYY-MM-DD }}-trends.md` with this format:

```markdown
# AI Trends — {{ date }}

## Top Trending Videos

| # | Title | Channel | Views | Link |
|---|-------|---------|-------|------|
| 1 | ... | ... | ... | [Watch](url) |

## Highlights

- Brief 1-2 sentence summary of the top 3 videos

---
*Generated by AI Trend Aggregator*
```

4. Ask the user how they'd like to receive the report: send via Gmail (use Gmail MCP), post to Slack (use Slack MCP), or save to Notion (use Notion MCP to create a new page titled "AI Trends — YYYY-MM-DD" under the configured parent page, with the report as the page content).
