Fetch today's trending AI videos and generate a report.

## Instructions

1. Use the YouTube MCP server to search for the top 10 trending AI videos from the last 24 hours. Search for topics like: "artificial intelligence", "machine learning", "LLM", "AI agents", "generative AI".

2. For each video, collect:
   - Title
   - Channel name
   - View count
   - Published date
   - URL
   - Brief description

3. Generate a clean markdown report and save it to `output/{{ YYYY-MM-DD }}-trends.md` with this format:

```markdown
# AI Trends — {{ date }}

## Top Trending Videos

| # | Title | Channel | Views | Link |
|---|-------|---------|-------|------|
| 1 | ... | ... | ... | [Watch](url) |

## Highlights

- Brief 1-2 sentence summary of the top 3 videos

---
*Generated by AI Trend Aggregator*
```

4. Save the report to Notion: Use the Notion MCP (`mcp__notion__API-post-page`) to create a new page under the parent page ID from the `NOTION_PARENT_PAGE_ID` env var. Pass `parent` as `{"page_id": "<NOTION_PARENT_PAGE_ID>"}` and `properties` with a title property set to "AI Trends — YYYY-MM-DD". Try including the report as `children` blocks; if that fails (serialization issue in v2.2.1), retry with title only and warn the user. If Notion fails entirely, warn but continue.

5. Ask the user if they also want to send the report via Gmail (use Gmail MCP) or post to Slack (use Slack MCP).
