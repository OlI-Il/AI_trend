Fetch today's trending AI videos and generate a report.

## Instructions

1. Use the YouTube MCP server's `search_trending_ai_videos` tool to find the top 10 trending AI videos from the last 24 hours. Search for topics: "artificial intelligence", "machine learning", "LLM", "AI agents", "generative AI".

2. For each video, collect:
   - Title
   - Channel name
   - View count
   - Published date
   - URL (`https://youtube.com/watch?v=VIDEO_ID`)
   - Brief description (first 100 chars)

3. Generate a clean markdown report and save it to `output/{{ YYYY-MM-DD }}-trends.md` with this format:

```markdown
# AI Trends — {{ date }}

## Top Trending Videos

| # | Title | Channel | Views | Link |
|---|-------|---------|-------|------|
| 1 | ... | ... | ... | [Watch](url) |

## Highlights

- Brief 1-2 sentence summary of the top 3 videos

---
*Generated by AI Trend Aggregator*
```

4. Ask the user if they want to send the report via Gmail. If yes, use the Gmail MCP to send the markdown report to the user's email.
